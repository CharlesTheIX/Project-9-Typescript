## Introduction

The repository uses `yarn` as its packages manager for `node.js` packages.

This repository uses the following languages / frameworks: 
- `/.scripts`
  - `Bash` (.sh)

- `/api`
  - `Typescrpt` (.ts)
  - `Express`

- `/cli`
  - `Typescrpt` (.ts)

- `/webapp`
  - `Typescript` (.ts)
  - `SASS` (.scss)
  - `Tailwind`
  - `NextJS`
